# 登录系统

这是SSO架构的中央登录页面，使用React和Vite构建。

## 功能特点

- 中央认证页面，为所有系统提供统一登录入口
- 登录后重定向回原始系统
- 自动检查现有认证状态
- 响应式设计，适配各种设备
- 用户注册功能
- 验证码保护
- 个人资料管理支持

## 使用技术

- React
- Vite
- Axios用于HTTP请求
- CSS3样式

## 页面功能

- 登录表单：用户输入凭据进行认证
- 注册表单：新用户创建账户
- 状态检查：自动检查用户是否已登录
- 重定向：登录成功后重定向回原始系统
- 验证码：防止自动化注册和登录
- 个人资料管理：支持用户注册时填写基本资料

## 环境要求

- Node.js (版本14或更高)
- SSO服务器正在运行

## 启动系统

```bash
npm install
npm run dev
```

系统将在端口3002上运行，访问地址：http://localhost:3002

## 工作流程

1. 用户访问需要认证的系统（如System A或System B）
2. 系统检测到用户未登录，重定向到登录页面
3. 新用户可以点击注册标签创建账户
4. 用户填写注册信息并输入验证码
5. 用户在登录页面输入凭据
6. 登录系统向SSO服务器发送认证请求
7. 认证成功后，重定向回原始系统
8. 原始系统验证令牌并显示受保护内容

## 安全特性

- 验证码保护防止自动化攻击
- 密码确认确保输入正确
- HTTPS支持（生产环境）
- 个人资料保护：用户只能访问和修改自己的资料
